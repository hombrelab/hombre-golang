The [hombre-golang](https://hub.docker.com/repository/docker/hombrelab/hombre-golang) image is based on the [official GO v1.16 Alpine 3v.12 image](https://hub.docker.com/_/golang).  
The image is used to create GO based [@Hombrelab](me@hombrelab.com) Docker images.  

Includes:
- linux packages: bash, nano, ca-certificates.
